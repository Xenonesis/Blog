## 🔧 Customization

### Colors
Edit `tailwind.config.js`:
```javascript
colors: {
    primary: { /* your colors */ }
}
```

### Fonts
Edit `resources/css/app.css`:
```css
@import url('your-font-url');
```

### Animations
Edit `tailwind.config.js`:
```javascript
animation: {
    'fade-in': 'fadeIn 0.5s ease-in-out',
}
```
