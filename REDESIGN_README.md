## 🔧 Customization

### Change Colors
Edit `tailwind.config.js`:
```javascript
colors: {
    primary: {
        500: '#YOUR_COLOR',
        600: '#YOUR_COLOR',
    }
}
```

### Change Fonts
Edit `resources/css/app.css`:
```css
@import url('your-google-font-url');
```

### Adjust Animations
Edit `tailwind.config.js`:
```javascript
animation: {
    'fade-in': 'fadeIn 0.5s ease-in-out',
}
```
